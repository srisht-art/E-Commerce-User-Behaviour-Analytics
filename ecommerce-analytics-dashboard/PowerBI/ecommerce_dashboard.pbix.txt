Power BI Dashboard File Placeholder

This is a placeholder for the Power BI dashboard file (ecommerce_dashboard.pbix).

To create the actual dashboard:
1. Open Power BI Desktop
2. Import the cleaned dataset from ../Data/cleaned_ecommerce_dataset.csv
3. Create visualizations as recommended in the main README.md
4. Save the file as ecommerce_dashboard.pbix in this folder

Recommended KPIs and Visualizations:
- Total Revenue, Total Orders, Conversion Rate, Cart Abandonment Rate, AOV, RPU
- Funnel Chart (User Journey)
- Bar Chart (Top Product Categories by Revenue)
- Line Chart (Monthly Revenue Trend)
- Demographic Breakdowns (Age, Gender, Location)
